# Reagent Addition

### Author
[Opentrons](https://opentrons.com/)



## Categories
* Sample Prep
	* Plate Filling

## Description

This protocol performs a simple reagent addition for WISTD and water to up to 96 samples in a 96-flat well plate.

---
