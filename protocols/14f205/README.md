# Biofilm Assay

### Author
[Opentrons](https://opentrons.com/)

## Categories
* Cell Culture
	* Assay

## Description
This protocol performs a custom biofilm assay using distilled H2O, crystal violet for staining, and 70% ethanol for final hydration. The assay is carried out after a user-defined incubation at 37ËšC using the Opentrons temperature module.

## Process
1. Input the mount side for your P300 multi-channel pipette and the number of samples to run.
2. Download your protocol.
3. Upload your protocol into the [OT App](https://opentrons.com/ot-app).
4. Set up your deck according to the deck map.
5. Calibrate your labware, tiprack and pipette using the OT App. For calibration tips, check out our [support articles](https://support.opentrons.com/en/collections/1559720-guide-for-getting-started-with-the-ot-2).
6. Hit "Run".
