## Description
Consolidate the contents of an entire 96 well plate into a single 1.5 mL microcentrifuge tube.

### Time Estimate
14 minutes

### Robot
* OT PRO 
* OT Standard
* OT Hood

### Modules


### Reagents


## Process
1. Pick up 20ul from wells until pipette is full
2. Dispense into tube
3. Repeat until plate is empty
